The Password Strength Checker is a beginner-friendly Python project that evaluates the strength of a user-entered password.
It checks whether the password follows essential security rules such as:

1)Minimum length

2)Presence of uppercase letters

3)Presence of lowercase letters

4)Presence of numbers

5)Presence of special characters


Based on these factors, the program labels the password as Weak, Medium, or Strong, and gives suggestions for improvement.

 Features

.) Checks password length
.) Detects lowercase characters
.) Detects uppercase characters
.) Detects numbers
.) Detects special characters
.) Gives suggestions to improve password strength
.) Beginner-friendly and easy to modify

How to Run

1. Install Python on your system.
2. Save the Python script (given in the project folder).
3. Open terminal/command prompt.
4. Run the program:
python password_strength_checker.py
5. Enter any password when prompted.
